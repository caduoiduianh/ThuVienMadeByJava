Books table:
```

    id VARCHAR(20) PRIMARY KEY,
    title VARCHAR(255),
    author VARCHAR(255),
    category VARCHAR(100),
    isbn VARCHAR(50),
    total_copies INT,
    available_copies INT
);
```

Readers table:
```
CREATE TABLE readers (
    id VARCHAR(20) PRIMARY KEY,
    name VARCHAR(255),
    phone VARCHAR(50),
    max_loans INT
);
```

Loans table:
```
CREATE TABLE loans (
    id VARCHAR(30) PRIMARY KEY,
    book_id VARCHAR(20),
    reader_id VARCHAR(20),
    borrow_date DATE,
    due_date DATE,
    return_date DATE,
    FOREIGN KEY (book_id) REFERENCES books(id),
    FOREIGN KEY (reader_id) REFERENCES readers(id)
);
```
